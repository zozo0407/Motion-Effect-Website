## 技术规范与约束

### 材质与光照

- **优先使用 MeshBasicMaterial**：不需要任何光照，适合大多数图像特效场景
- **ShaderMaterial 备选**：如果 MeshBasicMaterial 不能满足要求（如需要自定义着色器效果），则使用 ShaderMaterial 自定义材质和 shader
- **纹理设置**：材质的 side 必须使用 DoubleSide，避免背面剔除导致的闪烁或消失
- **纹理映射**：需明确是否默认填满0~1 UV范围；如需拆分/拼接，必须给出每个子物体的 uvScale 与 uvOffset

### 相机设置

- **相机类型**：必须使用透视相机（模板中已创建，FOV=53.1°，只读不可修改）
- **默认位置**：相机默认位置为 (0,0,10)
- **相机控制**：不使用轨道控制器（OrbitControls），不考虑键盘控制摄像机
- **相机位置计算规则**（确保无黑边、完整展示图片）：
  - 对于位置在 (x, y, z) 的**薄立方体**（size, size, 0.01）：
    - 相机位置：(x, y, z + size)
    - 看向目标：(x, y, z)
    - 此时薄立方体充满整个相机视角
  - 对于位置在 (x, y, z) 的**厚立方体**（size, size, depth）：
    - 相机位置：(x, y, z + size + depth/2)
    - 看向目标：(x, y, z)
    - 此时正立方体的正面充满整个相机视角（额外加的 depth/2 是立方体的+Z面距离原点的距离）
  - 一般情况下 size 为 10，如需明显前后关系，可将 size 设置更大，摆放距离相机更远，这样可以充当背景
- **相机描述要求**：需包含关键帧位置与 LookAt 目标点轨迹；如涉及路径运动，应给出每段的时间与缓动函数

### 动画与时间控制

- **时间单位**：统一使用毫秒(ms)
- **时间窗口**：所有动画需在预定义的 Duration 窗口内完成
  - 起始时间 ≥ 50ms
  - 结束时间 ≤ Duration - 50ms
  - 同一对象在同一时间段内不得被多个 Tween 同时操控
- **缓动函数**：
  - 允许集合：Sinusoidal（正弦）、Exponential（指数）、Circular（圆形）、Elastic（弹性）、Back（回弹）、Bounce（弹跳）、Quadratic（二次方）、Cubic（三次方）、Quartic（四次方）、Quintic（五次方）
  - **优先使用**：Sinusoidal、Exponential、Circular、Elastic、Back、Bounce
  - **低频率使用**：Quadratic、Cubic、Quartic、Quintic
  - **模式限制**：仅允许 In 与 InOut 模式，**禁止使用 Out 模式**
  - **连贯性要求**：动画应保持连贯性，禁止突变
- **Tween 动画注意事项**：
  - 动画之间不能有重叠的时间段操作同一个对象物体
  - 所有动画完成时间必须在 Duration 时间内，不能超过
  - 开始时间必须为 50ms 之后，结束时间必须为 Duration - 50ms

### 数值与单位

- **角度**：统一使用度(°)表示；涉及绕X/Y/Z轴旋转时，应明确轴向与正负方向；必要时给出关键帧的角度数值
- **位置与尺寸**：以世界单位表示，向量格式为 (x, y, z)
- **颜色**：以十六进制或 rgba 格式给出，优先使用十六进制（如 #FFFFFF、#000000）

### 3D对象与几何体

- **几何体类型**：平面、立方体、球体等
- **复杂对象**：如果一个复杂的对象由多个几何体组成，考虑组成一个 Group，后续调整复杂对象时可直接调整 Group
- **对象命名**：对象的名称需要涵盖自己的类型，不能有重复的名称
- **预设对象**：建议直接引用以下参数，以减少歧义
  - {PRESET_OBJECT}

## 特效考虑要点

在扩展和分析图像3D特效时，必须考虑以下方面：

### 图像特效方面

- 图像的3D变换（旋转、缩放、位移、扭曲）
- 图像分割和重组效果（破碎、分片、马赛克、像素化）
- 图像几何变形（弯曲、波动、拉伸、压缩）
- 图像材质和着色器效果（发光、透明度、颜色变化）

### 效果方面

- 两张或多张图像之间的过渡动画
- {TYPE}的时间轴和节奏控制
- {TYPE}的方向性和路径规划
- {TYPE}的中间状态和变化过程
- {TYPE}的视觉连贯性和流畅度

### 技术实现细节

- 图像加载和纹理映射到3D对象
- 3D几何体的创建和变形
- 动画的缓动函数和时间控制
- 相机的运动轨迹和视角变化
- **预设参考**（建议直接引用以下参数，以减少歧义）：
  - 相机运动预设：{PRESET_CAMERA_MOTION}
  - 渲染物体预设：{PRESET_RENDER_MOTION}
  - 图片A到图片B切换预设：{PRESET_TEXTURE_MOTION}

### 视觉效果

- 特效的视觉冲击力和吸引力
- 考虑使用高级 Tween 动画增强变换效果（参考上述动画与时间控制规范）
- 考虑用户体验和观看舒适度
