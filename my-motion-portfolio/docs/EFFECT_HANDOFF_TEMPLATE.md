### [B] 双缓冲模式 (Multipass / Feedback)
**适用场景**: 流体、烟雾、残影。
**逻辑**: 当前帧需要读取上一帧的渲染结果。
```mermaid

```
**注意**: 移动端移植时需开启 FrameBuffer Object (FBO) 并处理 Ping-Pong 交换逻辑。
